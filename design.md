# SCOOP 전동킥보드 주행기록 앱 디자인 문서

## 앱 개요

SCOOP은 전동킥보드 사용자를 위한 주행 기록 앱입니다. 실시간으로 속도, 거리, 시간을 측정하고 주행 히스토리를 관리할 수 있습니다.

## 화면 목록

### 1. 홈 화면 (Home)
메인 대시보드로 사용자의 주행 통계 요약과 빠른 주행 시작 버튼을 제공합니다.

### 2. 주행 화면 (Riding)
실시간 주행 중 표시되는 화면으로 속도계, 거리, 시간을 보여줍니다.

### 3. 히스토리 화면 (History)
과거 주행 기록 목록을 날짜별로 확인할 수 있습니다.

### 4. 프로필 화면 (Profile)
사용자 정보 및 앱 설정을 관리합니다.

## 주요 콘텐츠 및 기능

### 홈 화면
- 총 주행 거리 통계 카드
- 총 주행 시간 통계 카드
- 총 주행 횟수 통계 카드
- 주행 시작 버튼 (큰 원형 버튼)
- 최근 주행 기록 미리보기 (최대 3개)

### 주행 화면
- 현재 속도 (km/h) - 대형 숫자 표시
- 주행 시간 (HH:MM:SS)
- 주행 거리 (km)
- 평균 속도 (km/h)
- 최고 속도 (km/h)
- 일시정지/재개 버튼
- 정지 버튼

### 히스토리 화면
- 주행 기록 리스트 (FlatList)
- 각 항목: 날짜, 거리, 시간, 평균 속도
- 당겨서 새로고침

### 프로필 화면
- 사용자 닉네임
- 총 통계 요약
- 거리 단위 설정 (km/mile)
- 앱 버전 정보

## 핵심 사용자 플로우

### 주행 시작 플로우
1. 홈 화면에서 "주행 시작" 버튼 탭
2. 주행 화면으로 전환
3. 자동으로 주행 기록 시작
4. 일시정지 버튼으로 잠시 멈춤 가능
5. 정지 버튼으로 주행 종료
6. 주행 결과 요약 표시
7. 저장 후 홈 화면으로 복귀

### 히스토리 확인 플로우
1. 하단 탭에서 "히스토리" 탭 선택
2. 주행 기록 리스트 확인
3. 특정 기록 탭하여 상세 정보 확인

## 레이아웃 원칙

모든 화면은 모바일 세로 모드(9:16)를 기준으로 설계됩니다. 한 손 조작이 가능하도록 주요 버튼은 화면 하단에 배치합니다. iOS Human Interface Guidelines를 준수하여 네이티브 iOS 앱과 유사한 경험을 제공합니다.
