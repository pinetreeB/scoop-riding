# SCOOP 앱 장기 로드맵

## 비전
전동킥보드, 오토바이, 자동차 사용자를 위한 통합 모빌리티 플랫폼
- 주행 기록 및 분석
- 커뮤니티 기반 정보 공유
- 차량 관리 및 이력 추적
- 네비게이션 및 코스 추천

## Phase 2: 지도 기반 정보 공유 시스템 (다음 단계)

### 2.1 스탬프/깃발 시스템
사용자가 지도에 마커를 찍어 정보 공유

**마커 카테고리:**
| 카테고리 | 아이콘 | 설명 |
|---------|-------|------|
| 맛집 | 🍽️ | 라이더 추천 음식점 |
| 카페 | ☕ | 바이크 카페, 휴게소 |
| 주유소 | ⛽ | 고급유, 24시 주유소 |
| 화장실 | 🚻 | 공중화장실 |
| 편의점 | 🏪 | 24시 편의점 |
| 주차장 | 🅿️ | 무료/유료 주차장 |
| 위험구간 | ⚠️ | 단속카메라, 사고다발구역 |
| 뷰포인트 | 📸 | 경치 좋은 곳 |
| 와인딩 | 🛣️ | 추천 와인딩 코스 |

**기능:**
- 마커 생성/수정/삭제
- 사진 첨부
- 별점 및 리뷰
- 좋아요/저장
- 신고 기능

### 2.2 주행 구간 색상 표시
- 방문 횟수에 따라 도로 색상 변경
- 개인 히트맵 생성
- 전체 사용자 인기 구간 표시

### 2.3 GPX 기반 네비게이션
- GPX 파일 가져오기/내보내기
- 코스 따라가기 네비게이션
- 음성 안내 (선택적)

## Phase 4: 차계부 및 정비 이력 시스템

### 4.1 차계부 기능
**기록 항목:**
- 주유/충전 기록
- 정비 내역
- 튜닝 내역
- 소모품 교체 이력
- 사고 이력

**자동 알림:**
- 엔진오일 교체 주기
- 타이어 교체 주기
- 정기 점검 알림

### 4.2 차대번호(VIN) 기반 이력 관리
**핵심 기능:**
- 차대번호 등록 및 인증
- 출고일 자동 조회
- 누적 주행거리 추적
- 소유자 변경 이력
- 정비 이력 체인

**활용 시나리오:**
```
1. 중고차 구매자: 차대번호로 실제 주행거리, 사고이력 확인
2. 정비소: 차대번호로 이전 정비 내역 조회
3. 판매자: 투명한 이력 공개로 신뢰도 향상
```

### 4.3 정비소/딜러 연동
- 공식 정비소 등록
- 정비 기록 자동 연동
- 정비 예약 시스템
- 부품 가격 비교

## 기술 고려사항

### Google Maps API 전환 시점
**현재 (OpenStreetMap):**
- 무료, 제한 없음
- 기본적인 지도 기능

**Google Maps 전환 권장 시점:**
- 사용자 1,000명 이상
- 네비게이션 기능 필요 시
- 실시간 교통 정보 필요 시

**예상 비용 (사용자 10,000명 기준):**
- 월 $50-200

### 데이터베이스 확장
**현재 스키마에 추가 필요:**
```sql
-- 마커/스탬프 테이블
CREATE TABLE map_markers (
  id, user_id, category, lat, lng, 
  title, description, photos, rating, 
  created_at, updated_at
);

-- 차계부 테이블
CREATE TABLE vehicle_logs (
  id, vehicle_id, type, date,
  mileage, cost, description, photos
);

-- 차대번호 이력 테이블
CREATE TABLE vin_history (
  id, vin, event_type, user_id,
  mileage, description, verified, created_at
);
```
